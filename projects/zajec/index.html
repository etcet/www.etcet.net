In 1970, Edward Zajec, in collaboration with Matjaz Hmeljak, created The Cube: The and Variations (T.V.C.) on an IBM 7040 using FORTRAN IV. A language of 8 basic patterns and a set of syntactic rules is used to create a composition. These days you can do it real time. Algorithm not yet implemented. Left click cycles forward, right click backwards.

Examples: The Cube, Forms, ETCET
